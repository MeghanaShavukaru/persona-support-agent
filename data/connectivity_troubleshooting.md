# Connectivity Troubleshooting

## Network errors

If users experience connection timeouts, verify the local network firewall and DNS routing. Ensure outbound access to the API host is allowed.

## SSL/TLS issues

Confirm that the client trusts the certificate chain and that no proxy is intercepting HTTPS traffic.

## Recommended checks

- Ping the API endpoint
- Confirm the service endpoint URL
- Review proxy or VPN settings
