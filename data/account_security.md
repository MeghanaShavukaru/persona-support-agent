# Account Security

## Password rules

Passwords must be 12 characters or longer and include uppercase letters, lowercase letters, numbers, and symbols.

## Multi-factor authentication

We recommend enabling MFA for all admin users. If a login attempt is blocked, the user should verify their second factor and retry.

## Account lockout

After five failed login attempts, the account is locked for 30 minutes. An administrator can reset the lock from the account management console.
